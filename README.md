## Summary
This report analyzes four real phishing email samples. Each email was examined for red flags including fake sender domains, urgency tactics, generic greetings, and suspicious links. All four emails were classified as Phishing.

One email (Microsoft fake alert) was analyzed using MXToolbox Email Header Analyzer – see Appendix for screenshots.
